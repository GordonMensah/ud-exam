# UD Exam – Scripture Quiz & Exam Simulator

A Streamlit app for scripture-based MCQ quizzes and exams.

## Live App
Deploy via [Streamlit Community Cloud](https://share.streamlit.io)

## Local Run
```bash
pip install -r requirements.txt
cd quiz_app
streamlit run ui/streamlit_app.py
```

## Streamlit Cloud Deployment
1. Push this repo to GitHub
2. Go to https://share.streamlit.io
3. New app → select this repo
4. Main file path: `quiz_app/ui/streamlit_app.py`
5. Deploy
